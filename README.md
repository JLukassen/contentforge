# ContentForge

ContentForge is a Django-based content creation tool for drafting and formatting posts across platforms like X, Reddit, Instagram, and Telegram.

## Current Goal

Create one master post and generate platform-specific drafts.

## Planned Features

- Master post editor
- X post formatting
- Reddit title/body formatting
- Instagram caption formatting
- Hashtag tools
- Draft storage
- Manual copy/paste workflow
- Future API posting support
